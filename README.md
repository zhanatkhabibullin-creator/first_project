# 🚀 Three Axes Rescue

Учебный проект для практики работы с Git и GitHub.

## 📌 Описание

Этот проект создан в рамках обучения.  
Основная цель — освоить:
- работу с репозиториями
- коммиты
- ветки
- push/pull

Проект имитирует систему "спасения" по трём осям (X, Y, Z).

## 🛠 Технологии

- Python 3
- Git
- GitHub

## 📂 Структура проекта

    three-axes-rescue/
    │
    ├── main.py          # основной файл
    ├── rescue.py        # логика "спасения"
    └── README.md        # описание проекта

## ▶️ Запуск проекта

1. Клонировать репозиторий:

    git clone https://github.com/your-username/three-axes-rescue.git

2. Перейти в папку проекта:

    cd three-axes-rescue

3. Запустить:

    python main.py
